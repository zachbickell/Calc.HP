Calc.HP is a solo project by Zach Bickell. The entire project can be viewed on GitHub. The "Socialite" style was built from scratch in CSS. SCSS support is not integrated by default. All assets use the 'EB Garamond' font, which can be located in the Socialite file or on Google Fonts. All icons are courtesy of Font Awesome.
