General

This tool is a part of Calc.HP. This will allow you to calculate the resultant vector from 2 components

The standard method to calculate a resultant vector goes as follows:


2 directional components are given. One in the horizontal (x) direction, and one in the vertical (y) direction

to find the resultant vector, these 2 values are treated as legs of a right triangle. This makes it so that the resultant vector will be the hypotenuse of this triangle. this will allow us to use the Pythagorean Theorem to calculate the vectors magnetude.


Values can now be substituted in


Since vectors have both magnitude and direction, an angle must be calculated. This can be done using an inverse trigonometric function. looking back at the triangle, notice that the angle of the vector (hypotenuse) is opposite the Y component and adjacent to the X component. The tangent function relates these 2 sides, thus it must be used.



Info

Units must be user-entered. Linear direction of components must be specified by positive/negative symbols. For vertical (Y) component, leave positive for up/north, make negative for down/south. for horizontal (X), leave positive for right/east, make negative for left/west.
